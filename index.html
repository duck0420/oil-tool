📖 使用說明

🖊️ 每日登錄
點日曆上任一天 → 路線自動帶入預設模板 → 選交通工具 → 儲存。週末及國定假日自動跳過。

➕ 插入新停靠點
按「＋ 在此後插入」→ 新點起點自動接上一段終點 → 選迄點即可。

🚗 交通工具
每天選一次，所有路段共用。切換汽/機車時里程自動更新。預設汽車。

🏖️ 休假 / 請假
點日期旁的開關關閉 → 當天不算工作天、不列入統計。

📊 月結總覽
自動計算當月工作天、總里程、預估油資（費率可在 ⚙️ 調整）。

📥 匯出 / 列印
「📥 匯出」= 完整 Excel（外勤工作表＋申請差旅費，含簽名欄）。
「🖨️ 列印」= 直接開啟申請差旅費列印頁面。

☁️ 跨裝置同步
自動透過 Firebase 雲端同步。PC 改完 → 手機打開自動同步。
按 ☁️ 手動推送、⚙️ → 從雲端拉取（換裝置時用）。

👤 人員切換
右上角下拉可切換使用者。每人資料獨立、Firebase 分開儲存。
